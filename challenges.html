Challenges App

Challenges app is create-react-app PWA-application. Backend of this application have created using Amplify. Data storage is AWS DynamoDB and data will save using AWS AppSync GraphQl API. Authentication is created with AWS Cognito pool.

During this project I have learned how to use AWS console and how to create and host AWS-Amplify project.

User can create account and after that is possible to create a new challenges. Application counts how many days user have been successful with current challege.
