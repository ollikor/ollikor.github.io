Health app Github page

Api Github page

I have bathroom scales where I can get weight, fat percent and muscle percent and I wanted create Health app where I can set my body composition from my bathroom scales.

Layout have created using Bootstrap, data fetching from MongoDB via Express API. There is used Youtube API for getting videos to page. On the youtube video page have used redux to save channel what is selected so if user go to single video page and go back to youtube video page then application will fetch videos from same channel what was earlier.

On the body composition page will display all body compositions and there is possible to set new body composition. Body composition is possible also remove when you click some item on the graph.

On the food recipe page will display all recipes. There is possible to create new recipes and remove recipes. when you click some recipe application will move to single recipe page where is more information from recipe.

On the youtube videos page will display automatically videos from channel what is first channel on the database. There is possible to save new channel using channel id. There is possible to set channel which will display on the homepage. There is also possible to remove channel. When you click some video item, application will move to single video page.
